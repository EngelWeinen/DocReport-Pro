# DocReport Pro

&gt; AI-powered Word document summarizer with automatic email delivery.
&gt; 
&gt; AI 驱动的 Word 文档自动总结与邮件发送工具。

**by EngelWeinen**

---

## Features | 功能特性

- Select `.docx` files via GUI and extract full text automatically.
- Summarize documents using LLM APIs (Zhipu AI).
- Generate a clean summary report as a new Word document.
- One-click SMTP email delivery with attachment support.
- Real-time execution log with `[EW]` branding.

- 通过 GUI 选择 `.docx` 文件，自动提取全文。
- 调用大模型 API 进行文档总结（智谱 AI）。
- 生成格式整洁的 Word 总结报告。
- 一键 SMTP 邮件发送，自动附带报告文档。
- 实时日志输出，带 `[EW]` 标识。
